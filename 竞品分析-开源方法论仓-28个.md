## idea-to-product 定位

**竞争优势：**
- ✅ 三项标配全具备（Hard Gate + 快捷指令 + README 对比表）
- ✅ 专注"想法→产品"全链路，而非单一环节
- ✅ 中文友好，有完整中文文档

**差异化点：**
- 强制 7 步流程，比 superpowers 更结构化
- TDD + E2E 测试要求，比 get-shit-done 更严格
- 面向非技术用户，比大多数仓库更易上手
